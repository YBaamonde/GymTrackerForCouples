# GymTrackerForCouples

## Descripción
La aplicación GymTrackerForCouples es una herramienta web diseñada para que las parejas puedan hacer un seguimiento de sus rutinas de gimnasio, así como de su progreso y entrenamiento diario.

## Características
- Seguimiento de rutinas personalizadas
- Registro del progreso individual y en pareja
- Interfaz intuitiva y fácil de usar
